The Project

Neurons built from
discrete components.

Liminaut is a hobby-scale attempt to build a neuromorphic computing system from scratch, analog neurons made from salvaged and surplus components, built on recycled boards, figured out in real time.

The goal is something that can actually learn, using DIY memristive synapses and spike-timing-dependent plasticity built by hand. It's a slow project, openly documented, with no guarantee it works out the way I think it will.
